HISTORY

IBM wanted to set of documentation system in which we can mark the title ,heading,paragraphs and font type selection in the 1980s.They came out with a set of mark-up system called it Generic Mark Up Language(GML) In 1986,International Standardizing Orgnaization(ISO)took up this concept and standardized it as standardized Generlaized Mark Up language(SGML).In 1989 Tim Berners Lee and his team in the European Laboratory for particle Physics(CERR)de-signed the present form of the decomentation language and called it HTML.
